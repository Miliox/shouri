# CXX Project Template

Basic C++/CMake Template Project.

## Installation

Generate build files:

```
mkdir build
cd    build
cmake ..

make
```

### XCode

```
# In case of: No CMAKE_CXX_COMPILER could be found.
sudo xcode-select --reset

mkdir build
cd    build
cmake -G Xcode ..

xcodebuild
```
