# Shouri (勝利)

An Atari 2600 Emulator.

## Installation

```
git submodule update --init
```

## Make

```
mkdir build
cd    build
cmake ..

make
```

### XCode

```
# In case of: No CMAKE_CXX_COMPILER could be found.
sudo xcode-select --reset

mkdir build
cd    build
cmake -G Xcode ..

xcodebuild
```
